Скалярные переменные

Пока что во всех программах мы видели только встроенные текстовые строки. Давайте ощутим немного больше свободы, используя в программе переменные.

В Raku есть несколько типов переменных. Переменная — это контейнер, который может хранить в себе данные. Мы начнем с самых простых контейнеров, которые могут хранить в себе только одно значение. Такие контейнеры называются скалярами или скалярными контейнерами. Переменная имеет имя, в начале которого стоит особый индикатор, который говорит о сути контейнера. Этот индикатор — это символ, который называется сигил. Для скалярных контейнеров сигилом является знак доллара $.

Вот пара примеров скалярных переменных:

$name

$age
