#

Эта глава очень кратко описывает, как вы можете запустить программу, не покидая IDE, для двух самых популярных сред разработки.

Visual Studio Code Editor

Откройте терминал через меню Terminal → New Terminal (Терминал → Новый терминал). Это добавит панель под кодом программы. Выполните программу, как вы это делаете из командной строки:

$ raku hello.raku

Comma IDE

Средства запуска находятся в меню Run. Следуйте инструкциям по установке и настройке IDE на сайте Comma перед первым запуском программы.
