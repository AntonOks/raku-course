put

Per impostazione predefinita, la routine put svolge lo stesso lavoro di print ma aggiunge una nuova linea alla fine:

  1. Converte i suoi argomenti in una stringa chiamando il metodo Str su di essi.
  2. Aggiunge un carattere di nuova linea.
  3. Lo invia al flusso STDOUT.

Alcuni esempi:

42.put;
put 'Alpha', 'Beta';

my @array = 3, 4, 5;
put @array;

my %hash = a => 'b', c => 'd';
%hash.put;

Gli output del programma:

$ raku t.raku
42
AlphaBeta
3 4 5
a	b
c	d
