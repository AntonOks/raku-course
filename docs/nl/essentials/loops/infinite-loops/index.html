Oneindige lussen

De loop kan ook worden gebruikt met alleen een body:

loop {
    say 'Eternity';
}

Deze lus draait oneindig totdat je het programma beëindigt door bijvoorbeeld Ctrl+C in te drukken. (We zullen betere manieren hebben om de lus te controleren in het tweede deel van de cursus.)
