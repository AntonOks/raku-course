Palaišana no IDE

Šajā sadaļā ļoti īsi tiek izskaidrots, kā palaist programmu, neatstājot IDE, divām visvairāk ieteicamajām IDE.

Visual Studio Code redaktors

Atveriet termināli no izvēlnes Terminal → New Terminal. Tas pievienos paneli zem jūsu programmas teksta. Palaidiet programmu tāpat, kā to darāt no komandrindas:

$ raku hello.raku

Comma IDE

Palaišanas funkcija atrodas aiz izvēlnes vienuma Run. Sekojiet instrukcijām Comma vietnē, kā iestatīt IDE pirms pirmās programmas palaišanas.
