Tipu konvertēšana tipizētajiem mainīgajiem

Tipu ierobežojumi ir pietiekami stingri. Piemēram, jūs nevarat piešķirt Num vērtību, pat ja tā satur veselu skaitli, mainīgajam, kas deklarēts kā Int:

my Int $x = 42;
# $x = 42e1; # Kļūda

Izmantojiet vienu no metodēm, lai pārveidotu vērtību.

my Int $x = 42;
$x = 42e1.Int;
$x = Int(42e1);
