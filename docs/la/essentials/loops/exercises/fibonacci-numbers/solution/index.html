Solutio: Fibonacci numeri

Algorithmus sequens numeros Fibonacci definit.

  1. F0 = 0, F0 = 1
  2. Fn = Fn-1 + Fn-2

Codex

Codex algorithmum litteratim implet. Utiur duobus variabilibus ad numeros Fibonacci actuales conservandos et eos in circulo renovat. Nota quod ambo variabiles renovantur in una assignatione.

my $a = 0;
my $b = 1;

say $a;
for ^19 {
    ($a, $b) = $b, $a + $b;
    say $a;
}
