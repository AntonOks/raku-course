put

Per defaltam, put munus idem officium facit ac print sed addit novam lineam in fine:

  1. Convertit argumenta sua ad stringam vocando methodum Str in eis.
  2. Addit characterem novae lineae.
  3. Mittit ad STDOUT rivum.

Pauca exempla:

42.put;
put 'Alpha', 'Beta';

meus @array = 3, 4, 5;
put @array;

meus %hash = a => 'b', c => 'd';
%hash.put;

Exitus programmatis:

$ raku t.raku
42
AlphaBeta
3 4 5
a	b
c	d

Nova linea put

Veri characteres qui adduntur post exitum sumuntur ex methodo nl-out rivi exitus. Valor defaltus est \n.
