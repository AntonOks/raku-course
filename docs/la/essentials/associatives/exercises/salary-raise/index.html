Stipendium incrementum

Problema

Datis tabula cum notitia operarii:

my %employee =
    full-name => 'Eliza Vents',
    department => 'Sales',
    salary => 3000;

Exemplum

Hic est expectatus exitus programmatis:

$ raku salary-raise.raku
New salary of Eliza Vents: 3150
