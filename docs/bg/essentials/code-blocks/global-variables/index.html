Глобални променливи

Нека създадем променлива преди блока с код:

my $x = 1;

{
    $x = 42;
    say $x;
}

say $x;

В този случай, $x е глобална променлива (намира се в глобалния обхват), и програмата отпечатва 42 два пъти.
