Running from IDE

This section very briefly explains how you run the program without leaving the IDE for the two most recommended IDEs.

Visual Studio Code Editor

Open a terminal from the menu Terminal ‚Üí New Terminal. It will add a panel below your program text. Run the program as you do it from the command line:

$ raku hello.raku

Comma IDE

The running facility sits behind the Run menu item. Follow the instructions at the Comma website on how to set up the IDE before the first run of the program.
