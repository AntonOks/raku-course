Lösung: Wahre und falsche Zeichenketten

Diese Übung ähnelt der vorherigen, aber diesmal ist die Aufgabe etwas kniffliger. Es gibt mehr ‘verdächtige’ Zeichenketten, die als False behandelt werden können. Lassen Sie uns diese erkunden.

say ?'';

say ?' ';
say ?'        ';

say ?"\n";

say ?'0';
say ?'0.0';

Von all diesen wird nur die leere Zeichenkette als False gewertet. Alle anderen Zeichenketten, selbst diejenigen, die nur Leerzeichen oder das explizite Zeichen 0 enthalten, werden zu True.
