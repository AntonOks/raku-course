Funktionsnamen

Der Name einer Funktion ist ein Bezeichner, der denselben Regeln wie Variablennamen folgt. Er muss mit einem Buchstaben oder Unterstrich beginnen und kann von beliebigen Buchstaben, Ziffern, Unterstrichen, einfachen Anführungszeichen (Apostroph) und Bindestrichen gefolgt werden.

Apostrophe und Bindestriche müssen von alphabetischen Zeichen gefolgt werden. Während Apostrophe in Bezeichnern nicht weit verbreitet sind, werden Bindestriche häufig verwendet, da sie ein zusätzliches Maß an Ausdruckskraft bieten.

Nachfolgend ist eine kurze Liste von Beispielen für gültige Funktionsnamen aufgeführt.

greet

greet-them

hello-world

NumberOfItems

is-valid

to_be-or-not_to_be
