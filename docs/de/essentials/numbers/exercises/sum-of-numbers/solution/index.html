Lösung: Summe der Zahlen

Hier ist die Lösung für das Problem.

Code

my $a = prompt 'Geben Sie die Zahl A ein: ';
my $b = prompt 'Geben Sie die Zahl B ein: ';
my $c = prompt 'Geben Sie die Zahl C ein: ';

my $sum = $a + $b + $c;

say "Die Summe der Zahlen ist $sum.";
