## Problēma

Uzrakstiet programmu, kas lūdz lietotājam ievadīt veselu skaitli un saglabā to tipētā mainīgajā. Izdrukājiet ievadīto skaitli.

## Piemērs

Šī programma darbojas šādi:

```console
$ raku input-number.raku
Ievadiet skaitli: 42
42
```
