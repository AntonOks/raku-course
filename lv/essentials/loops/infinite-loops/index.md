{% include menu.html %}

`loop` var izmantot arī tikai ar ķermeni:

```raku
loop {
    say 'Eternity';
}
```

Šī cilpa darbojas bezgalīgi, līdz jūs pārtraucat programmu, piemēram, nospiežot `Ctrl+C`. (Mums būs labāki veidi, kā kontrolēt cilpu kursa otrajā daļā.)
