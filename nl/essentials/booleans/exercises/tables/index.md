## Probleem

Maak een programma dat de resultaten van de drie Booleaanse operaties afdrukt voor alle combinaties van `True` en `False`.

## Voorbeeld

Hier is een voorbeeld van gedeeltelijke uitvoer van dit programma.

```console
$ raku tables.raku
True && True is True
True && False is False
. . .
```
