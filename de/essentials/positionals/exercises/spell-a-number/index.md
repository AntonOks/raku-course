## Problem

Schreiben Sie ein Programm, das eine positive ganze Zahl von der Kommandozeile entgegennimmt und sie auf Englisch buchstabiert. Lösen Sie das Problem für Zahlen unter 100.

## Beispiel

```console
$ raku spell-a-number.raku 34
thirty-four
```
