## Problema

Scrivi un programma che chiede all'utente di inserire un numero intero e lo salva in una variabile tipizzata. Stampa il numero inserito.

## Esempio

Questo programma funziona nel seguente modo:

```console
$ raku input-number.raku
Enter a number: 42
42
```
