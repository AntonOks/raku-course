---
title: Глобальные переменные
---

{% include menu.html %}

Давайте создадим переменную до начала блока кода:

```raku
my $x = 1;

{
    $x = 42;
    say $x;
}

say $x;
```

В этом случае `$x` — глобальная переменная (она находится в глобальной области
видимости), и программа выводит `42` дважды.
