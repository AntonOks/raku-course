## Задача

Вам дано число, которое жестко закодировано в программе. Выведите его дробную часть. Вот начало программы:

```raku
my $n = 15.8972;
# ... Ваш код здесь
```

## Пример

Программа должна вывести часть числа после десятичной точки:

```console
$ raku fractional-part.raku
0.8972
```
