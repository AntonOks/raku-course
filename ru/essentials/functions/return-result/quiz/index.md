## 1

Что возвращает эта функция?

```raku
sub f($a, $b) {
    return $a + $b if $a > $b;
    return 42;
}
```

{:.quiz-select}
42 | `f(1, 2)` возвращает&nbsp; (: 1, 2, 3, 42 :)
3 | `f(2, 1)` возвращает&nbsp; (: 1, 2, 3, 42 :)
42 | `f(2, 2)` возвращает&nbsp; (: 1, 2, 3, 42 :)

## 2

Что возвращает эта функция?

```raku
sub g() {
    return -1;
    1;
}
```

{:.quiz-select}
−1 | `g()` возвращает&nbsp; (: −1, 1 :) | Последнее выражение `1;` никогда не достигается.
