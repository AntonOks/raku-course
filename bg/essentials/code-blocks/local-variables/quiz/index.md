## 1

Какво отпечатва тази програма?

```raku
my $value = 100;

{
    my $value = 1;
    $value *= 2;
    say $value;
}
```

{:.quiz-select}
2 | Отговор: (: 1, 2, 100, 200 :) | Използва се и се отпечатва локална променлива.

## 2


{:.quiz-select}
100 | Отговор: (: 1, 2, 100, 200 :) | Локалната променлива е модифицирана, но глобалната променлива е отпечатана.
