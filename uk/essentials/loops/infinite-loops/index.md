`loop` також може використовуватися лише з тілом:

```raku
loop {
    say 'Вічність';
}
```

Цей цикл працює нескінченно, доки ви не завершите програму, наприклад, натиснувши `Ctrl+C`. (Ми матимемо кращі способи контролю циклу в другій частині курсу.)
