## Проблема

Напишіть програму, яка запитує у користувача введення цілого числа та зберігає його в типізованій змінній. Виведіть введене число.

## Приклад

Ця програма працює наступним чином:

```console
$ raku input-number.raku
Enter a number: 42
42
```
